## Castone Design personal idea 프로토타입
- 2026.07.14 제작 시작
- TinyML을 이용한  제스처 인식, 비접촉 컨트롤러 

- 손목 근육의 전기신호(EMG)를 애플워치같은 밴드형 기기가 읽어 손동작을 알아채고, 그 판단을 밴드 안의 초소형 머신러닝으로 즉시 처리하여 노트북, 스마트홈 같은 스마트 기기를 손짓 하나로 조작하는 비접촉 컨트롤러

- 감지: EMG 센서
- 원리: 손목 근육의 전기신호를 읽어 제스처 판별
- 처리: TinyML(초소형 머신러닝)로 MCU(마이크로컨트롤러유닛)에서 실시간 분류
- 출력: GPIO로 조명켜기 -> BLE HID-노트북/TV, WIFI-스마트 홈 조작

- 설계 팁: 제스처↔명령 매핑을 코드에 작성하기 보다는 설정으로 분리하고, "모드 전환 제스처"를 하나 두면 PC 모드 ↔ TV 모드 ↔ 집 모드를 손짓으로 전환할 수 있습니다. Mudra(무드라) 밴드가 쓰는 방식인데, 이걸 구현하면 "범용성"을 강하게 어필할 수 있습니다.

- team meeting 후 난이도 문제로 인해 프로토타입 만들어보았다.
- Xcode를 설치하였고 애플워치를 착용하여 내장된 가속도계로 팔의 움직임 크기를 재서, 임계값을 넘으면 흔들기로 인식하는 앱
